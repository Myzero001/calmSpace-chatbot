Calm Space Chatbot 🌿

“พักใจสักนิด… แล้วไปต่อด้วยรอยยิ้ม”

Calm Space Chatbot คือแชตบอทที่ถูกออกแบบมาเพื่อเป็น “พื้นที่ปลอดภัยทางใจ” สำหรับผู้ใช้งาน โดยเน้นการรับฟัง ให้กำลังใจ และช่วยสะท้อนความรู้สึกอย่างอ่อนโยน เพื่อช่วยลดความเครียด ความกังวล และเพิ่มแรงใจในการใช้ชีวิตประจำวัน

## ✨ Features

* 💬 พูดคุยแบบเพื่อนที่หวังดี
* 🤍 รับฟังโดยไม่ตัดสิน
* 🌱 ให้กำลังใจและคำแนะนำแบบเรียบง่าย
* 🧠 จำบริบทการสนทนา (Memory)
* 📚 ใช้ RAG ดึงข้อความให้กำลังใจจากฐานข้อมูล
* 🔍 ใช้ Vector Store ด้วย FAISS + HuggingFace Embeddings
* 🛡️ มีระบบป้องกันการให้คำแนะนำทางการแพทย์

## 🧠 System Overview

ตัวบอทจะรวมข้อมูลจาก:

* Context จาก RAG
* Context จาก Memory
* ข้อความปัจจุบันของผู้ใช้

เพื่อสร้างคำตอบที่มีเหตุผล เข้าใจอารมณ์ และตอบสนองอย่างเป็นธรรมชาติ

---

## 🛡️ Safety

Calm Space Chatbot ถูกออกแบบมาเพื่อ “รับฟังและให้กำลังใจ” เท่านั้น
ไม่สามารถใช้แทนแพทย์หรือผู้เชี่ยวชาญด้านสุขภาพจิตได้

หากพบข้อความเสี่ยง ระบบจะแนะนำให้ผู้ใช้งานพูดคุยกับผู้เชี่ยวชาญ
